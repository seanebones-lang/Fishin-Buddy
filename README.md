**BiteCast** turns data into thrilling bite predictions! Predict probabilities, find hot spots, get frenzy alerts – feel the rush like a pro angler. Cross-platform iOS/Android with cinematic UI, ML accuracy >85%, gamified streaks & badges.

![Hero](https://via.placeholder.com/800x400/0066cc/ffffff?text=BiteCast+-+Hook+Your+Next+Big+Catch) <!-- Replace with splash mock -->

## 🚀 Quick Features
- **Bite Index**: Hourly 0-100% predictions (bass, trout, etc.) via NOAA/USGS ML.
- **Hyper-Local**: Top spots, routes (Mapbox), bait shops.
- **Alerts**: Push 'Frenzy!' haptics & voiceovers.
- **Log & Share**: Catches, badges, community brags.
- Offline-ready, freemium.
